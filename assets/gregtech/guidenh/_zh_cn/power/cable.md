# 导线与线缆

> [!NOTE]
> 延伸阅读: "[GTNH中文维基 - 导线与线缆](https://gtnh.huijiwiki.com/wiki/导线与线缆)"。

**导线与线缆**（电线）是在 [局部电力网络](enet.md) 中连接输电方块和用电方块输送电力的媒介。它们本身并不缓存电力。

电线有三个参数，均显示在 Tooltip 中。

# 最大电压与最大电流

**最大电压**只和电线材料有关。电线有 1x、2x、4x、8x、12x、16x 的变种，成倍率地扩大材料决定的基础最大电流。

- 最大电压较低的电线无法传输更高电压的能量包。承载电压或电流超过该值时，电线会烧毁。
- 最大电压较高的电线可以正常传输更低电压的能量包。

电流是可叠加量——某格导线承载的电流是所有流经它的电流之和。某格电线是否会烧毁取决于所有流经它的能量包中电压最高者。

# 线损率

使用电线输电时，路径上的电流值处处相同，而电压值可能因 [线损](cable-loss.md) 而下降。

- 线损率与电线的粗细、电流大小无关，仅与材料和是否包层（导线 / 线缆）有关。线缆的线损往往只有对应导线的一半。
- 部分电线的线损率为 0，能实现超导。如 1x 红石合金线缆、1x MV 超导导线和 1x 无尽导线等。

# 替换与染色

手持一种导线与线缆，<kbd>Shift</kbd> + <kbd>左键</kbd>已放置的另一种导线与线缆，可以直接替换，保留各方向的连接状态与覆盖板。替换后原来与设备相连的电线会断开连接。

可使用喷漆罐为电线染色，有助于在不同的局部电力网络中辨别承载不同电压的电线。
